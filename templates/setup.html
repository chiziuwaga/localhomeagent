Let's configure your Local Home Agent

Welcome!

This wizard will help you set up your Local Home Agent for managing your co-living space.

You'll configure:

  • Admin PIN and passphrase (required first)
  • Home name and basic settings
  • Local AI model for offline operation
  • Home Assistant integration (optional)

Choose Your Admin Credentials

The agent generated a temporary bootstrap PIN and saved it to ~/.local-home-agent/FIRST_RUN_CREDENTIALS.txt in your home folder. Open that file once, then pick your own PIN and passphrase below.
