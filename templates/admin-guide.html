🏠 WiFi Network Setup for Co-Living

This guide explains how to configure the Local Home Agent so that all residents can access it via their devices on your home WiFi network.

✓ How It Works

The Local Home Agent runs on your Windows computer and serves a web interface on your local network. Any device connected to the same WiFi can access the dashboard, chat with the AI, and control smart home devices.

📋 Prerequisites Checklist

  • Windows 10/11 computer that will host the agent
  • Computer connected to your home WiFi
  • Administrator access on the computer
  • Know your WiFi network name (SSID)
  • All residents' devices connected to the same WiFi

1 Find Your Local IP Address

First, you need to find the IP address of the computer running the Local Home Agent. This is the address other devices will use to connect.

Option A: Using Command Prompt

  1. Press Win + R, type cmd, press Enter
  2. Type ipconfig and press Enter
  3. Look for "IPv4 Address" under your WiFi adapter
  4. It will look like: 192.168.1.xxx or 10.0.0.xxx

2 Configure Windows Firewall

Windows Firewall may block incoming connections. You need to allow the Local Home Agent through the firewall.

⚠️ Security Note

Only allow connections on your private (home) network. Never expose this to public networks.

Allow Through Firewall

  1. Open Windows Security (search in Start menu)
  2. Go to Firewall & network protection
  3. Click Allow an app through firewall
  4. Click Change settings (requires admin)
  5. Click Allow another app...
  6. Browse to LocalHomeAgent.exe and add it
  7. Check Private network only

Alternative: PowerShell Command

Run PowerShell as Administrator and execute:

New-NetFirewallRule -DisplayName "Local Home Agent" `
    -Direction Inbound -Protocol TCP -LocalPort 8000 `
    -Action Allow -Profile Private

3 Share Access with Residents

Once configured, residents can access the Local Home Agent from any device on the WiFi network.

Add to Home Screen (PWA)

For the best experience, residents can add the Local Home Agent to their phone's home screen:

  1. iOS Safari: Tap Share → Add to Home Screen
  2. Android Chrome: Tap Menu (⋮) → Add to Home Screen
  3. The app will work offline and feel like a native app!

4 Add Residents as Users

Each resident should be added as a user with appropriate permissions.

User Roles

  • Admin: Full control - manage users, settings, all devices
  • Resident: Control allowed devices, view status, chat with AI
  • Guest: Limited access - view only, basic controls
👥 MANAGE USERS

5 Security Best Practices

Recommended Settings

  • ✓ Set a strong admin PIN for high-risk actions (unlocking doors, etc.)
  • ✓ Enable action verification for security-sensitive operations
  • ✓ Keep the host computer updated with Windows security patches
  • ✓ Use a strong WiFi password (WPA3 preferred)
  • ✓ Regularly review the audit log for unusual activity
⚠️ Important

Never share the access URL publicly. Keep it within your home network only. The Local Home Agent should only be accessible on your private WiFi.

🔧 Troubleshooting

Can't connect from other devices?

  1. Verify both devices are on the same WiFi network
  2. Check Windows Firewall is allowing the connection
  3. Try temporarily disabling firewall to test (re-enable after!)
  4. Make sure the host computer isn't in sleep mode
  5. Verify the Local Home Agent is running

Connection is slow?

  1. Check WiFi signal strength on both devices
  2. Move closer to the router
  3. Consider a WiFi extender for larger homes
